# Emily AI: PM Agent
A Streamlit application that utilizes Groq and Notion APIs to assist with project management and idea generation.

## Features
- Supports multiple languages (Korean and English)
- Integrates with Groq API for AI-powered idea generation and code snippets
- Utilizes Notion API for database management and task creation
- Allows users to select project management methodologies (Agile, Scrum, Kanban, Waterfall)
- Generates social media threads based on project ideas

## Requirements
- Streamlit
- Groq API
- Notion API
- Python 3.8+

## Usage
1. Install required libraries using `pip install -r requirements.txt`
2. Set up Groq API key and Notion token in the sidebar
3. Select project management methodology and language
4. Enter project idea and description
5. Click "Start Emily PM" to generate project plan and code snippets
6. Optionally, generate social media threads
